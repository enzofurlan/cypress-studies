## Description

Project created to practice Cypress skills.

## Getting Started

### Dependencies

* Docker
* Node.js 12 or 14 and above

### Installing

* For M1 Macs - make sure to:
```
export DOCKER_DEFAULT_PLATFORM=linux/amd64
```
```
docker-compose up -d
```
You might need to wait or run again.

### Executing program

* Cypress basic commands
```
cypress npx open //or
cypress npx run
```
